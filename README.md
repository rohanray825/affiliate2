# Affiliate App Directory

Next.js affiliate promotion website with an embedded Sanity Studio and Cloudinary URL-based image management.

## Setup

1. Copy `.env.example` to `.env.local`.
2. Add your Sanity project ID, dataset, and write token.
3. Add Cloudinary delivery URLs to app and site records in Sanity Studio at `/studio`.
4. Run `npm run dev`.

Without Sanity credentials, the website uses a built-in sample catalog.
